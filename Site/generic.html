Estratégias

Objetivo é até o final do curso ter um jogo 2D criado.
Irei utilizar como estratégia, uma separação em etapas, sendo elas:

1. aprimoramento da minha ideia
2. O modelo de negócios.
3. O Game Designer Document totalmente feito
3. A base do jogo feita, contendo a movimentação o personagem e o mapa
4. Um diagrama de sistema UML
5. o script todo funcionando
6. Por fim o jogo depois de passado por testes pronto.
